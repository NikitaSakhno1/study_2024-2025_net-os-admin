Студент: Сахно
Группа: нфибд-02-23

Цель

Приобрести практические навыки по установке и конфигурированию DNS-сервера, усвоить принципы работы системы доменных имён.

Задания

Установите на виртуальной машине server DNS-сервер bind и bind-utils.
Сконфигурируйте на виртуальной машине server кэширующий DNS-сервер.
Сконфигурируйте на виртуальной машине server первичный DNS-сервер.
При помощи утилит dig и host проанализируйте работу DNS-сервера.
Напишите скрипт для Vagrant, фиксирующий действия по установке и конфигурированию DNS-сервера во внутреннем окружении виртуальной машины server. Соответствующим образом внесите изменения в Vagrantfile
